How to Play

Overview

This game uses your browser's developer tools. Use Inspect Element and Console to find hidden keys embedded in the page's HTML, CSS, or JS. Then type the key into the overlay to score points.

Hints

  • Look for data-key attributes on native HTML tags (section, article, li, etc.).
  • Search HTML comments for .
  • Check the Console for console.log hints (inside the simulated page).
